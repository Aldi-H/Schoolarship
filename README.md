## Run Backend

1. Install Golang, lihat panduan: https://go.dev/doc/install
2. Clone repo ini.
3. Masuk ke folder <b>backend</b> (`cd backend/`)
4. Jalankan command: `go mod tidy`
5. Jalankan server: `go run main.go`
6. Lihat endpoint pada log.
7. Done.
